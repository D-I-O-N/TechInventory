# TechInventory
WPF-Приложение для для учета электронно-вычислительной техники. В него включены: диаграммы, работа с файлами (CSV, Word, Excel), работа с БД (SQL) через SQL Server Management, дизайн форм.
WPF is an application for accounting of electronic computing equipment. It includes: diagrams, working with files (CSV, Word, Excel), working with databases (SQL) through SQL Server Management, form design.
